What is UP-WARD?

UP-WARD is the fastest and easiest way to add a return button to the top of the page to your website. This library is supported by Chrome, Safari, Firefox and Edge browsers.

Installing

You can quickly install UP-WARD via npm : 

                $ npm install --save @ripecherries/up-ward

Or, if you want something a little more lightweight, you can give Yarn a try: 

                $ yarn add --dev @ripecherries/up-ward
