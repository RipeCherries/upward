Quick Start

Adding your first return button to the top of a web page is very simple. All you need is to write just 2 lines of code: 

                import UpWard from "@ripecherries/up-ward";

const up = new UpWard();

And that’s it! Do you want to change the appearance of the button? Just pass the UserSettings object to the constructor: 

                const up = new UpWard({
    position: "right",
    offset: 30,
    bottomOffset: 30,
    containerSize: 50,
    containerColor: "#17252A",
    arrowColor: "#FEFFFF",
    trigger: 800,
    zIndex: 1
});

For a full list of options, see Options & Configuration
