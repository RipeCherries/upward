Options & Configuration

UP-WARD options:

Option Description Type Default
position On which side the button will be shown ("left" or "right") String left
offset How many pixel the button is distant from the edge of the screen, based on the location setted Integer 20
bottomOffset How many pixel the button is distant from the bottom edge of the screen Integer 20
containerSize The width and height of the button Integer 30
containerRadius Allows you to round the corners of the button Integer 10
containerColor The color of the container (in HEX format) String #000000
arrowColor The color of the arrow (in HEX format) String #FFFFFF
alwaysVisible Set to true if u want the button to be always visible Boolean false
trigger After how many scrolled down pixels the button must be shown Integer 100
zIndex Set the z-index Integer 1
